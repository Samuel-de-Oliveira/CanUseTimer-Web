The lightweight software for SpeedCubing

The CanUseTimer is a Opensource SpeedCubing timer. But this one
only is terminal interface program, who made it more lightweight and
softest than outhers Speedcubing timers, totaly created for who only
want simplicity or have a bad hardware PC or computer (And it's perfect
for who use the i3 window manager). This program is ported for every
platform (Windows, Linux, MacOS) and the source is open for study or
upgrades.

Why use?
  • Lightweight program (Almost 120Kb of space in Linux);
  • Run all platforms (Windows, Linux, MacOS X);
  • Don't need internet (but you can't update it);
  • It follow the WCA regulations;
  • Open source (Read the GPLv3 licensse for more info).
